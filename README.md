# Etch-a-Sketch

This is a simple Etch-a-Sketch web app built as part of [The Odin Project's Foundations Course](https://www.theodinproject.com/paths/foundations/courses/foundations).

## 🔧 Features

- 16x16 default drawing grid using `div`s and Flexbox.
- Hovering over grid squares changes their color (initially black).
- Button to resize the grid up to 100x100 squares.
- Random RGB coloring effect on hover.
- Fully responsive grid maintained within 960x960 space.

## 🚀 How to Use

1. Hover over the squares to draw.
2. Click the **"Resize Grid"** button to set a new size (max 100).
3. Watch as the squares light up with random colors!

## 💡 What I Learned

- DOM manipulation using JavaScript.
- Creating elements dynamically.
- Using Flexbox for grid layouts.
- Adding event listeners and handling user input.
